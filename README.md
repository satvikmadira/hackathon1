# 📊 InsightAI — Free BI Dashboard (Powered by Google Gemini)

A fully browser-based AI-powered BI dashboard — **completely free to use!**
Type plain English questions, get instant interactive charts. No backend, no credit card.

**[🔗 Live Demo →](https://YOUR-USERNAME.github.io/insightai)**

## ✨ Features

- 💬 **Natural language → charts** powered by Google Gemini AI (FREE)
- 📁 **CSV upload** — query any dataset instantly
- 📊 **Smart chart selection** — line, bar, pie, doughnut, horizontal bar
- 🎯 **Metric cards** with trend indicators
- 💡 **AI insight summaries** per query
- 🔐 **Login screen** with demo accounts
- 📜 **Query history** per session
- 🚀 **Zero backend** — pure HTML/CSS/JS, deploy anywhere

## 🆓 Get Your Free Gemini API Key

1. Go to **aistudio.google.com**
2. Sign in with your **Google account** (Gmail)
3. Click **"Get API Key"** → **"Create API key"**
4. Copy the key — it starts with `AIzaSy...`
5. Paste it in the app under **⚙️ Settings**

**Free limits:** 15 requests/minute · 1,500 requests/day · No credit card!

## 🚀 Deploy to GitHub Pages (Free Hosting)

### Step 1 — Create GitHub account
Go to [github.com](https://github.com) and sign up.

### Step 2 — Create new repository
1. Click **+** → **New repository**
2. Name it `insightai`
3. Set to **Public**
4. Click **Create repository**

### Step 3 — Upload files
1. Click **"uploading an existing file"**
2. Drag and drop `index.html` and `README.md`
3. Click **Commit changes**

### Step 4 — Enable GitHub Pages
1. Go to **Settings** tab
2. Click **Pages** in left sidebar
3. Source → **Deploy from a branch**
4. Choose **main** → **/ (root)** → **Save**

### Step 5 — Your site is live!
```
https://YOUR-USERNAME.github.io/insightai
```

## 🔒 Security Note

Your API key is stored **only in your browser session** (JavaScript memory).
It is never written to localStorage, cookies, or any server.
It disappears when you close the tab or log out.
